Ограничения (упрощения) программы.
1. Товары могут быть только штучными. Например, нельзя продать 1.5 кг.
2. Сохранённые чеки не содержат итоговых сумм и стоимостей товаров. Следовательно, при изменении цены в будующем старые чеки отобразятся не верно
3. Нет обработок ошибок ввода в методе OnAddOrderPosion
4. Нет механизма приёма наличных и рассчёта сдачи покупателю.
5. Использована eager loading стратегия для загрузки данных из базы данных
6. Сервис для работы с базой данных загружает всю базу данных, вместо страничной загрузки.
7. Вкладка "История" не оформлена т.к носит информативный характер для отладки
